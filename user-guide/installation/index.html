Installation

Packages

You can download ScalaMock jars and sources from Sonatype.

Sources

You can find the ScalaMock repository and source code on Github.

sbt projects

To use ScalaMock in sbt with ScalaTest add the following to your project file:

libraryDependencies += "org.scalamock" %% "scalamock-scalatest-support" % "3.2.2" % "test"

To use ScalaMock with Specs2:

libraryDependencies += "org.scalamock" %% "scalamock-specs2-support" % "3.2.2" % "test"

Maven projects

To include ScalaMock in your Maven project add the following to your pom.xml:

<dependency>
    <groupId>org.scalamock</groupId>
    <artifactId>scalamock-scalatest-support_2.11</artifactId>
    <version>3.2</version>
    <scope>test</scope>
</dependency>
