Reporting Bugs

ScalaMock bugs can be submitted to our issues page on github. Please note that this is not the right place to ask for help or general questions about how to use ScalaMock.

Finding Help

You can ask questions and seek help on Stack Overflow with #scalamock tag.

Gitter Chat

While ScalaMock does not have its own chat channel, it is worth asking in the official Gitter Scala channel which is well attended and pretty helpful.
